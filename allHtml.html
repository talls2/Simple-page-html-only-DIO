Tags HTML

Para que servem?

As tags html foram criadas para a organização do HTML, fazendo assim uma linguagem de marcação mais facil de ser escrita e lida

Exemplos de tags e seus usos:

  • Tag "ul" e "li": utilizadas para criar listas não ordenadas
  • tags "i" são utlizadas para deixar um trecho em italico, "strong" para deixar em negrito, e "small" para deixar trechos pequenos
  • A tagi "u" é utilizada normalmente para sublinhar palavras escritas de forma errada. a tag sup pode ser utilizada para fazer um texto superescrito a tag "blockquote" é utilizada quando queremos fazer uma sitação direta de outro website, exemplo:
    fazer seu bolo de cenoura de liquidificador de uma vez só é preciso um liquidificador potente, mas se achar a massa pesada para o seu aparelho, corte a cenoura em pedaços bem pequenos e só bata no aparelho os ingredientes úmidos. Depois que estiver tudo homogêneo, incorpore a mistura líquida aos ingredientes secos, mexendo bem e delicadamente.
